### Universal Auto-Data Analyst
### A smart application that cleans, analyzes, and visualizes any dataset instantly.

### Features:
Agnostic: Drag and drop a CSV or Excel file, the AI detects column types automatically.

Auto-Cleaning: Automatic management of missing values and date formats.

Smart Insights: Automatic detection of correlations and anomalies (Outliers).

Interactive: Dynamic dashboard with Plotly and Streamlit.

### Installation:
pip install -r requirements.txt

streamlit run app.py

With automatic analysis:
streamlit run app_with_analyzer.py
